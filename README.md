# Circus

A web-based recreation of the classic Atari 2600 game **Circus Atari** (1980).

## Play the Game

Open `index.html` in a modern web browser, or serve locally:

```bash
npx serve .
```

## How to Play

- Use **left/right arrow keys** or **mouse** to move the seesaw
- Bounce the clowns upward to pop balloons
- Higher balloons are worth more points
- Don't let the clowns hit the ground!

## About the Original

Circus Atari was released by Atari in 1980, inspired by the 1977 Exidy arcade game "Circus". Players control a teeter-totter to launch clowns at three rows of balloons. The game features satisfying physics and increasingly challenging gameplay as balloons are cleared.
